--> CLASES MÚLTIPLES: este elemento tiene DOS clases asignadas al mismo tiempo, separadas por un espacio dentro del mismo atributo class. Esto hace que coincida con AMBAS reglas simultáneamente: - p.quitar --> aporta color: red; - .importante --> aporta font-size: 20px; Como ambas reglas afectan a PROPIEDADES DISTINTAS (color y font-size), no hay conflicto entre ellas: el navegador combina ambos estilos y el párrafo se muestra en rojo Y con tamaño 20px al mismo tiempo.
